## 🛍️ Laivaly - Clothing Brand Backend
Laivaly is a full-stack e-commerce application for a clothing brand. This is the backend server built with Node.js, Express, TypeScript, and MongoDB.

### 🚀 Tech Stack
- Node.js, Express.js
- TypeScript
- MongoDB + Mongoose
